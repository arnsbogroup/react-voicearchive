# react-voicearchive [![Build Status](https://travis-ci.com/ArnsboMedia/react-voicearchive.svg?token=n7HfC44DzNaKKMLUT1pH&branch=master)](https://travis-ci.com/ArnsboMedia/react-voicearchive) [![codecov](https://codecov.io/gh/ArnsboMedia/react-voicearchive/branch/master/graph/badge.svg?token=qW7SrPfHta)](https://codecov.io/gh/ArnsboMedia/react-voicearchive)

react-voicearchive is a [React](https://reactjs.org) package using [styled components](https://www.styled-components.com/) to centralize components in a VoiceArchive design.

## Requirements

- [NodeJS](https://nodejs.org)
- [Yarn](https://yarnpkg.com/lang/en/)

## Installation

To install this package in a React project, by using Yarn:

```shell
$ yarn add https://github.com/ArnsboMedia/react-voicearchive.git
```

or by using npm:

```shell
$ npm install --save https://github.com/ArnsboMedia/react-voicearchive.git
```

## Usage

To import a component in your project:
```js
import { Card } from 'react-voicearchive';
```

Please refer ot `src/components` to get a list of available components and their props.

## Build

You can build the package by using yarn:

```shell
$ yarn build
```

or by using npm:

```shell
$ npm run build
```
